# DrumCharter Resources

This directory contains reference materials and templates for the DrumCharter project.

## Files

- `song-chart-template.md`: Standard markdown-based template for new song charts.
- `drum-symbols-guide.md`: Reference for all supported drum symbols and their meanings.
- `example-have-it-all.md`: Complete example chart for the song "Have It All".
- `example-center.md`: Complete example chart for the song "Center".
